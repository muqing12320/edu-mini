## 零、函数

### 1. 函数

**定义**：从一个集合（定义域）到另一个集合（值域）的**唯一确定**的映射关系 $f: X \to Y$。

**视角**：这是最底层的概念，只关心“给一个 $x$，得一个 $y$”。

### 2. 向量值函数

**定义**：输入一个实数 $t$，输出一个向量 $\mathbf{v}$。

**形式**：$\mathbf{r}: \mathbb{R} \to \mathbb{R}^n$

**联系**：

- 参数方程 $\begin{cases} x = \cos t \\ y = \sin t \end{cases}$ **等价于** 向量值函数 $\mathbf{r}(t) = \langle \cos t, \sin t \rangle$。
- **参数方程本身不是函数**（因为自变量是隐式的），但由它**构造**出的映射是函数。
- 这是**复值函数**在二维实空间上的**几何同构体**（见下文对比）。

### 3. 复值函数

**定义**：只要**值域是复数集 $\mathbb{C}$** 的函数。定义域可以是实数，也可以是复数。

**子分类**：

- 若自变量是**实数** $t$：$f(t) = e^{it} = \cos t + i\sin t$ （实质等同于二维向量值函数）。
- 若自变量是**复数** $z$：$f(z) = z^2$ （这就是**复变函数**）。

### 4. 复变函数

**定义**：**定义域和值域都是复数域 $\mathbb{C}$** 的函数。$f: \mathbb{C} \to \mathbb{C}$。

**核心区别**：
虽然复数 $z = x+iy$ 看起来像二维向量 $(x,y)$，但**复变函数**比**二维向量值函数**多了**解析性**（可导性）的强约束。

| 对比项       | **二维向量值函数** $\mathbf{F}(x,y)$         | **复变函数** $f(z)$             |
| :----------- | :------------------------------------------- | :------------------------------ |
| **写法**     | $\mathbf{F} = (u(x,y), v(x,y))$              | $f = u(x,y) + i v(x,y)$         |
| **乘法**     | 无自然乘法（只有点积/叉积）                  | 有复数乘法结构                  |
| **可导条件** | 偏导数存在即可（弱）                         | 必须满足**柯西-黎曼方程**（强） |
| **结论**     | 复变函数是**具有额外代数结构**的二维向量场。 |                                 |
