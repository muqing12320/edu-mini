## 配置数据库连接

![image-20260323095736133](./img/%E4%B8%9A%E5%8A%A1%E6%95%B0%E6%8D%AE%E6%A2%B3%E7%90%86/image-20260323095736133.png)



已在`hosts`文件中配置`node1`的地址映射。如未配置，也可直接使用`MySQL Server`所在节点的`IP`。

### Windows Hosts配置（管理员权限）

1. 按 `Win + R` → 输入 `notepad` → **Ctrl + Shift + Enter**（管理员运行）

2. 记事本 → 文件 → 打开 → 粘贴路径：
   ```
   C:\Windows\System32\drivers\etc\hosts
   ```

3. 添加地址映射，如：

   ```bash
   192.168.107.101 node1
   ```

4. 修改后直接保存即可。
